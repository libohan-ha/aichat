项目提示与密钥示例（不要放真实密钥到仓库）

DeepSeek（聊天使用）
- API Base: https://api.deepseek.com/v1
- API Key: DEEPSEEK_API_KEY=sk-your-deepseek-key
- Model: DEEPSEEK_MODEL=deepseek-chat

Gemini（用于“AI起草”等可选功能）
- API Key: GEMINI_API_KEY=AIzaSy-your-gemini-key
- HTTPS 代理（可选）: GEMINI_PROXY=http://127.0.0.1:7890

本地开发建议
- 将真实密钥写入 .env.local（已在 .gitignore 中忽略）
- 允许局域网访问开发静态资源：ALLOWED_DEV_ORIGINS=http://你的局域网IP:端口

注意
- 请勿提交 ai.txt 或任何包含真实密钥的文件到 GitHub。
- 本文件仅作为示例与备忘。
